## 🚀 Featured Projects

### 🔹 Arabic Academic RAG System
> *Making university regulations instantly searchable in Arabic*

- **Problem:** Students couldn't quickly find answers in dense Arabic academic documents
- **Solution:** Built a full RAG pipeline — hybrid TF-IDF + keyword retrieval over a structured `data.json` canonical source, powered by a local Ollama LLM
- **Stack:** `Python` `LangChain` `Ollama` `TF-IDF` `Arabic NLP`
- **Impact:** Reduced document lookup time from minutes to seconds; works fully offline with no API cost

---

### 🔹 Fruit Classification Conveyor System
> *Computer vision meets real hardware automation*

- **Problem:** Manual fruit sorting is slow, inconsistent, and error-prone
- **Solution:** Designed an end-to-end system — CNN model for classification, Arduino UNO for motor control, servo for sorting gates, Python inference pipeline bridging vision to hardware
- **Stack:** `TensorFlow` `Keras` `OpenCV` `Python` `Arduino` `L298N`
- **Impact:** Achieved automated real-time sorting; system generalizes across fruit types via swappable models

---

### 🔹 University Portal Sync Automation (UniSync)
> *Automating the tedious parts of student life*

- **Problem:** Manually downloading lecture files from the university portal daily was time-consuming
- **Solution:** Built a Python automation script — scrapes the portal, syncs new files to Google Drive and local disk, sends desktop notifications, runs daily via Task Scheduler
- **Stack:** `Python` `BeautifulSoup` `Requests` `Google Drive API v3` `Plyer`
- **Impact:** Zero manual intervention; 100% of new files captured automatically every day

---

### 🔹 Data Analysis Internship Projects @ NTI
> *Real-world data work at a national institution*

- **Problem:** Raw telecom and business datasets with no structure or story
- **Solution:** Delivered end-to-end analysis — data cleaning, EDA, feature engineering, and Power BI dashboards for stakeholder reporting
- **Stack:** `Python` `Pandas` `NumPy` `Power BI` `Excel` `SQL`
- **Impact:** Produced KPI dashboards used for internal decision-making; developed data cleaning pipelines that reduced preprocessing time significantly
