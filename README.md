### 🟢 RAG — KSUAI Academic System `NEW`
> Arabic RAG system for the Faculty of AI academic regulations at Kafr El-Sheikh University.  
> Ask any question in Arabic about study rules, graduation requirements, and course plans.  
> Built with **Hybrid TF-IDF + Keyword Retrieval** and a local **Ollama LLM**.

🔗 [mhmd-ebrahim-1/RAG_KSUAI](https://github.com/mhmd-ebrahim-1/RAG_KSUAI) · `Python` · `RAG` · `Arabic NLP` · `Ollama`
